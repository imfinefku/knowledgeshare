# 知识分享刷题小程序

## 项目简介

知识分享刷题小程序是一个集知识分享以及刷题于一体的微信小程序。<br>
本系统分为小程序端和管理端，小程序端采用微信原生开发语言，后端采用SpringBoot+layui。<br>
知识分享刷题小程序非常贴合学校和学生的需求，题目经典不失新颖，且界面美观大方，非常适合作为毕业设计或者学习使用。<br>

## 安装教程

1.安装jdk1.8。<br>
2.安装MySQL5.7。<br>
3.在mysql中创建数据库knowledgeshare，导入后台代码中的knowledgeshare.sql。<br>
4.在idea中导入后端项目knowledgeshare-java。<br>
5.在微信开发者工具中导入微信小程序项目knowledgeshare-wx。<br>
6.启动后端程序默认启动端口是10000，可自行修改。<br>
